Form Validation

Form validation checks whether the data entered by the user is valid or not. If the entered data is correct, the form is submitted to the server. Otherwise, an error message is displayed.

Common HTML Validation Attributes:

  • required – Makes a field mandatory.
  • min – Minimum value.
  • max – Maximum value.
  • pattern – Specifies a regular expression.
  • step – Specifies the interval for numeric values.
  • maxlength – Maximum number of characters.
  • minlength – Minimum number of characters.

Common Input Types:

  • type="email"
  • type="url"
  • type="number"
  • type="date"
  • type="time"
  • type="datetime-local"
  • type="month"
  • type="week"
